# Hướng dẫn & Ngữ cảnh cho Gemini Agent

File này dùng để lưu trữ các prompt, rules và ngữ cảnh của dự án FSchool dành cho AI trợ lý Gemini (Google).

## Tech Stack
- **Frontend (Mobile):** Flutter (lib: dio, go_router). State Management: setState.
- **Backend:** .NET 8 Web API. Database: SQL Server. ORM: Entity Framework Core.
- **Architecture (Backend):** 3-layer (Controllers, Services, Repositories).

## Các quy tắc chung khi code
1. Không thay đổi kiến trúc nếu không có yêu cầu.
2. Viết code sạch, kèm comment giải thích những đoạn phức tạp.
3. Luôn sử dụng tiếng Anh cho việc đặt tên biến/hàm, và tiếng Việt cho giao diện người dùng.
