# Hướng dẫn & Ngữ cảnh cho Claude Agent

File này dùng để lưu trữ các prompt, rules và ngữ cảnh của dự án FSchool dành cho AI trợ lý Claude (Anthropic).

## Tech Stack
- **Frontend (Mobile):** Flutter (lib: dio, go_router). State Management: setState.
- **Backend:** .NET 8 Web API. Database: SQL Server. ORM: Entity Framework Core.
- **Architecture (Backend):** 3-layer (Controllers, Services, Repositories).

## Các quy tắc chung khi code
1. Bám sát Tài liệu SRS và cấu trúc thư mục đã thống nhất.
2. Viết code tối ưu hiệu năng, tuân thủ nguyên tắc SOLID cho .NET.
3. Đảm bảo UI/UX của Flutter đi theo guideline màu sắc: Cam (#FF6B00) và Trắng (#FFFFFF).
